Usage

  • You can click in the element to see more information about it.

  • You can also navigate straight to a specific element by typing its symbol or name in the address bar like this:

      or





    Make sure to type /#/ before the name or symbol.
  • If you click in the List of elements you can have the elements in the form of a table.



    You can sort any of the columns and export to .csv

Information about the content

  • The colors used in the periodic table are the CPK-colors. [Reference]
  • The colors used in the Atomic Number, Symbol and Name (black or white) don't follow any standard. They are calculated so they contrast with the background.
  • All the info about the Chemical Elements is from Bowserinator's github repository forked to my github account.
